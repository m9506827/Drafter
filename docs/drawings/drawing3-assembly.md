# Drawing 3: 完整組合施工圖

## 概述

完整組合施工圖顯示整個軌道系統的組合視圖，包含所有直段、彎段與腳架。

## 軌道路徑側視圖

### 上軌路徑
- 從左到右連續繪製
- 包含直段與彎段
- 彎段以弧線表示

### 下軌路徑
- 與上軌平行
- 相同的直段/彎段配置

### 腳架
- 在直段中點位置
- 垂直連接上下軌
- X 標記表示交點
- 底座繪製

### 尺寸標註
- 各段長度
- 軌道間距
- 彎管角度

## 繪製邏輯

### 路徑重建
1. 從取料明細重建軌道路徑
2. 直段：水平線段
3. 彎段：圓弧
4. 依序連接

### 腳架放置
1. 計算各直段中點位置
2. 在中點位置繪製腳架
3. 標註球號

### 尺寸標註
1. 各段長度
2. 軌道間距
3. 彎管角度（178°）

## 生成程式

相關程式碼位於 `generate_sub_assembly_drawing()` 中的 Drawing 3 區塊。

主要函數：
- `_draw_rail_path()` - 繪製軌道路徑
- `_draw_cutting_list_table()` - 繪製取料明細表
- `_draw_bom_table()` - 繪製 BOM 表
