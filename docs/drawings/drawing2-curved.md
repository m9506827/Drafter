## 仰角計算

仰角從 `track_elevations` 取得：
1. 遍歷所有軌道仰角
2. 取第一個非零值
3. 排除彎管（仰角=0）

```python
elevation_deg = 0
for te in track_elevations:
    angle = te.get('angle_deg', 0)
    if angle > 0:
        elevation_deg = angle
        break
```
