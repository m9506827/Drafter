## 側視圖

### 上軌（Upper Rail）
- 從左到右依序繪製
- 包含直段與轉折彎
- 紅色中心線
- 管壁雙線

### 下軌（Lower Rail）
- 與上軌平行偏移
- 軌道間距標註

### 腳架（Legs）
- 垂直於軌道
- X 標記表示與軌道交點
- 底座繪製

### 尺寸標註
- 各段長度
- 軌道間距
- 彎管角度

## 取料明細表

位於右上方，包含：

| 欄位 | 說明 |
|-----|------|
| 球號 | U1, U2, U3... (上軌) / D1, D2, D3... (下軌) |
| 取料尺寸 | 管徑 + 長度 或 角度(半徑)弧長 |

### 直管段格式
```
U1    直徑48.1 長度221.3
```

### 彎管段格式
```
U2    直徑48.1 角度12度(半徑270)外弧長62
```
